部署说明：

1. 请下载 xiguan-daka.zip 文件
2. 登录 GitHub，访问 https://github.com/gitjianyusu/xiguan-daka
3. 点击 'Add file' -> 'Upload files'
4. 将 xiguan-daka.zip 中的文件拖拽上传
5. 点击 'Commit changes'
6. 等待几分钟后，访问 https://gitjianyusu.github.io/xiguan-daka/ 即可查看网站

注意：
- 网站主文件是 index.html
- 清除计算数据的工具是 clear_calc_data.html
- 上传时会保留现有的 git 历史
